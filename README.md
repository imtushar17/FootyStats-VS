# FootyStats VS

A premium static web dashboard for comparing national football teams with side-by-side metrics, visual charts, tactical overlays, and match simulation.

## Features

- Team comparison interface with interactive selectors
- Dynamic visual metric bars
- Tactics pitch and squad visualization
- Trophy cabinet and match simulator UI
- Light/dark theme and animated ambient effects

## Files

- `index.html` — Main HTML page
- `style.css` — Styling and layout
- `script.js` — Interactive behavior and data logic

## Run locally

1. Open `index.html` in a browser.
2. Alternatively, serve the folder with a simple local server if desired.

## Recommended GitHub Pages setup

This is a static site, so it can be published directly with GitHub Pages from the repository root.

## Notes

- No build step is required.
- All assets are included in the repository.
